User Guide

 Assay-Based Incidence Estimation (ABIE) v3:

Incidence_Prevalence_Calculator

 

It is recommended that users of the Incidence_Prevalence_Calculator workbook read through this calculator-specific user guide before using the calculator.

 

In addition to this workbook-specific guide, users will need to familiarise themselves with the overall ABIE user guide/version 3 release note, which can be found on a separate tab <About ABIE v3> in this workbook. The overall ABIE guide provides important contextual information, including suggested further reading, relevant to the correct use of this tool; in particular, critical information on

  • nuanced concepts around input requirements and the meaning of outputs, and

  • licensing and citation.

Summary

The Incidence_Prevalence_Calculator estimates HIV incidence and prevalence from a cross-sectional survey which tested subjects for HIV and followed up by testing for biomarkers of recent infection amongst HIV positives. Required data comprises:

  • sample counts

  • recency test characteristics

  • estimates of “design effects” for the precision of cluster based sampling estimates of a) HIV prevalence, and b) the proportion of “recent” infections among HIV positives tested for recency

Worksheets in this Workbook

The Incidence/Prevalence Calculator contains six worksheets or tabs:

  • <About ABIE v3>: introduction to the ABIE toolbox in general and major release three

  • <User Guide>: this document

  • <Single Calculation>: incidence from a sing cross-sectional survey

  • <Scenario Comparison>: multiple (up to 10) sets of inputs with graphical summary

  • <FRR Sensitivity Analysis>: incidence confidence interval (CI) as a function of FRR

  • <Global DE Sensitivity Analysis>: incidence CI as a function of a global design effect

 

Each sheet is described in more detail below.

<Single Calculation>

Context:

A cross-sectional survey has been executed with a recency test whose properties (MDRI and FRR) have been estimated. Both HIV status and recent/non recent infection status amongst HIV positives have been ascertained (not necessarily for all subjects).

 

Utility:

This sheet calculates point estimates and 95% confidence intervals for prevalence and incidence.

 

Note:

  • Subjects whose HIV status was left unascertained are nowhere accounted for in this analysis.

  • The calculation accounts for HIV positive subjects whose recent/non-recent status was unascertained.

  • Incidence (including a 95% confidence interval) is reported both as an “instantaneous rate”, with units of % per annum (p.a.), and as a cumulative annual risk of infection.

  • Warnings about negative incidence estimates are indeed “warnings” and not “errors”, even though, physically, incidence of course takes strictly non-negative values.

  • The relative standard error of the incidence estimate is not reported if the point estimate is less than or equal to zero.

<Scenario Comparison>

Context:

A cross-sectional survey has been executed with a recency test whose properties (MDRI and FRR) have been estimated, but questions remain about the local applicability of these estimates. Both HIV status and recent/non recent infection status amongst HIV positives has been ascertained (not necessarily for all subjects).

 

Utility:

This sheet accesses the survey results entered into the <Single Calculation> sheet, while allowing variable specification of test properties – for example, leading to a sensitivity analysis indicating how the incidence estimate depends on the assumed value of MDRI or FRR.

 

Note:

  • Each scenario is specified in a column much like column D in the <Single Calculation> sheet. For consistency of layout, the recycled survey data is reproduced in each column.

  • It is recommended that the user at some point enter each input set into the <Single Calculation> sheet, in order to view possible relevant warnings and error messages.

  • It is envisaged that the main use of this sheet is to implement sensitivity analysis of incidence estimates as a function of one test property at a time; the 10 scenarios can however be selected arbitrarily and independently of each other.

  • It would be inappropriate to try to use this sheet to consider how incidence estimates would vary for different recent/non-recent discrimination thresholds on a given test. This is because the sheet reuses a single set of counts of recent, non-recent, and HIV negative results from the <Single Calculation> sheet, and a changing threshold would also change these counts.

<FRR Sensitivity Analysis>

One of the most common uses of the <Scenario Comparison> sheet is likely to be a sensitivity analysis considering dependence of the incidence estimate on the estimate of FRR (and this should always be investigated at some point for any data set). The <FRR Sensitivity Analysis> sheet thus automatically implements this analysis by using input from the <Single Calculation> sheet to plot the incidence estimate and 95% confidence interval against a hypothetical FRR. Changes to input on the <Single Calculation> sheet automatically carry through to the figure. Intermediate results required to populate the figure are found on a hidden sheet, which may be unhidden without a password (home tab, cells toolbar, format button → drop down menu: hide & unhide → secondary dropdown, unhide sheet).

 

Note:

  • The x variable is a hypothetical FRR, assumed to be known precisely.

  • The range of FRR values shown is from 0 to twice the point estimate entered in the <Single Calculation> sheet.

<Global DE Sensitivity Analysis>

This sheet displays the incidence estimate and 95% CI as a function of a global scaling of the variance of the incidence estimate by a hypothetical overall “design effect” ranging from 1 (no effect) to 5. The figure is linked to the inputs of the <Single Calculation> sheet, and is populated by intermediate results found on a hidden sheet (which may be unhidden without password).
